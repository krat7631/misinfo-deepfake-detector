# AI Internship Showcase Checklist

Use this before sharing on LinkedIn, GitHub, or in applications.

## Must-Have Signals

- Problem statement and impact are clear in `README.md`.
- Architecture is documented in `docs/ARCHITECTURE.md`.
- Reproducible setup works from scratch using `.env.example`.
- Tests and CI pass on the default branch.
- Model evaluation report is present and recent.

## Demo Assets

- 60-90 second demo video:
  - text detection example
  - image/video example
  - explanation response
- 3 screenshots:
  - input form
  - API response
  - evaluation metrics artifact

## Talking Points for Interviews

- Why this model choice?
- What are failure modes?
- How do you prevent abuse or unsafe uploads?
- How would you scale video inference?
- What would you improve with more time?
